# pauv-docs

Docs for Pauv, built with [Mintlify](https://mintlify.com).

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mintlify) to preview the documentation locally:

```bash
npm i -g mintlify
```

Run the following command at the root of the repository (where `mint.json` lives):

```bash
mintlify dev
```

## Publishing

Changes pushed to the default branch are automatically deployed via [Mintlify](https://mintlify.com).
